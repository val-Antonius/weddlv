# Product Overview - weddlv

## Project Purpose
weddlv is a modern, self-service SaaS platform that enables couples to create, customize, and distribute beautiful, interactive digital wedding invitations with seamless mobile experience.

## Value Proposition
- **Digital-First Approach**: Eliminates traditional paper invitations with eco-friendly digital alternatives
- **Self-Service Platform**: Empowers couples to create professional invitations without design expertise
- **Interactive Experience**: Provides RSVP management, guestbook, and event details in one platform
- **Mobile-Optimized**: Ensures perfect viewing experience across all devices

## Key Features & Capabilities

### Core Features
- **Invitation Creation**: Multi-step wizard for creating custom wedding invitations
- **Template System**: Pre-designed templates with customization options
- **RSVP Management**: Integrated RSVP collection and tracking system
- **Guest Management**: Comprehensive guest list and response tracking
- **Digital Guestbook**: Interactive guestbook for guest messages
- **Event Details**: Multiple event support with detailed information
- **Photo Gallery**: Wedding photo sharing capabilities

### Technical Capabilities
- **Real-time Updates**: Live RSVP tracking and guest management
- **Secure Authentication**: User account management with Supabase Auth
- **Data Validation**: Comprehensive form validation with Zod schemas
- **Responsive Design**: Mobile-first design approach
- **SEO Optimized**: Server-side rendering with Next.js App Router
- **Email Integration**: Automated email notifications via Resend

### Administrative Features
- **Admin Dashboard**: Comprehensive invitation management interface
- **Analytics**: RSVP tracking and guest response analytics
- **Publishing Control**: Draft/publish workflow for invitations
- **Bulk Operations**: Efficient management of multiple invitations

## Target Users

### Primary Users
- **Engaged Couples**: Planning their wedding and seeking modern invitation solutions
- **Wedding Planners**: Professionals managing multiple client weddings
- **Event Coordinators**: Organizing wedding-related events

### Use Cases
- **Wedding Invitations**: Primary digital invitation distribution
- **Save the Dates**: Early wedding announcements
- **Event Management**: Multiple wedding events (ceremony, reception, etc.)
- **Guest Communication**: Centralized guest interaction platform
- **RSVP Collection**: Streamlined response management

## Current Development Status
**Phase 0 Complete**: Foundation established with Next.js 16, TypeScript, Supabase integration, and UI component library setup.

**Phase 1 In Progress**: Core MVP development including authentication, RSVP functionality, and admin dashboard.
